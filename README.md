# Hand Gesture Recognition using OpenCV & MediaPipe

This project detects hand gestures in real-time using Python.

## Features
- Real-time hand tracking
- Gesture detection
- Sound effects (Rasengan, Chidori, Kamehameha)
- Particle animations

## Technologies Used
- Python
- OpenCV
- MediaPipe
- Pygame

## How it works
- Detects hand using MediaPipe
- Recognizes gestures
- Triggers sound and animation
